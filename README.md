🚐 VanCash — فان كاش
تطبيق مبيعات متنقّلة كامل للبيع بالجملة والتجزئة، يعمل بدون إنترنت، بواجهة عربية، مصمّم للسوق الجزائري.
ملف واحد · يُثبَّت كتطبيق · بياناتك محليّة على جهازك فقط.
---
✨ المميزات
🛒 بيع سريع — نقدي / آجل / جزئي، مع تعديل السعر ومنح تخفيضات
🧾 فاتورة عربية + طباعة بلوتوث حرارية (58mm / 80mm)
👥 الزبائن والديون — كشف حساب كامل، تحصيل تلقائي على الفواتير الأقدم
🏷️ المنتجات والمخزون — إضافة/تعديل، تحميل وجرد مع كشف العجز
📊 التقارير — مبيعات، ربح، أفضل المنتجات (يوم/أسبوع/شهر)
✅ التصفية اليومية — مطابقة النقد المتوقع بالفعلي وكشف الفروقات
💸 المصاريف — لحساب الربح الصافي
💾 نسخ احتياطي ثلاثي — داخلي تلقائي + تذكير ذكي + حفظ في ملف
🎨 3 أنماط ألوان — نهار / ليل / صحراء
📴 أوفلاين 100% — يعمل بدون أي اتصال
---
📲 التثبيت كتطبيق
افتح الرابط في Chrome على أندرويد
اضغط ⋮ (ثلاث نقاط) → تثبيت التطبيق
ستظهر أيقونة «فان كاش» على شاشتك الرئيسية
---
🛠️ التقنيات
HTML + CSS + JavaScript (بدون أُطر ثقيلة)
Dexie.js فوق IndexedDB للتخزين المحلي
Web Bluetooth + ESC/POS للطباعة الحرارية
محاسبة بنظام القيد المزدوج
PWA (Service Worker + Manifest) للعمل أوفلاين
---
🔒 الخصوصية
كل البيانات (الزبائن، المبيعات، الديون) تُخزَّن محليًا على جهازك فقط.
لا شيء يُرفع إلى أي خادم. المستودع يحتوي على الكود فقط.
> 💡 صدّر نسخة احتياطية بانتظام من: **الإعدادات → تصدير نسخة احتياطية**
